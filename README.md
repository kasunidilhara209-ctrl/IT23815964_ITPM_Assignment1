~ Project Overview
This repository contains the automated Playwright test scripts for testing the Swift Translator system ([https://www.swifttranslator.com/](https://www.swifttranslator.com/)).  
The purpose of this project is to verify how accurately the system converts Singlish input into Sinhala output and to evaluate the usability and stability of the user interface.

~ Features
- 24 Positive Functional Test Cases (correct conversion scenarios)  
- 10 Negative Functional Test Cases (incorrect conversion scenarios)  
- 2 Positive UI Test Cases (real-time output behavior)  
- 2 Negative UI Test Cases (UI-related failure scenarios)  
- Covers various sentence structures, grammar forms, input lengths, slang, punctuation, and mixed-language inputs

~ Prerequisites
- Node.js (v18 or higher)  
- npm (comes with Node.js)  
- Playwright

~ Project Structure
- tests/
  - itpm_assignment.spec.js
- playwright.config.js
- package.json
- README.md
~ How to Run Tests
1. Open terminal in project folder
2. Run:
   npx playwright test
